# VPN-бизнес под ключ

Полный комплект для запуска автоматизированного VPN-сервиса на Telegram.  
**Бот принимает оплату → выдаёт ключи → отправляет напоминания — без ручного труда.**

🌐 **Живой сайт:** https://irekgai111-lang.github.io/vpn-sell/

---

## Файлы проекта

| Файл | Назначение |
|---|---|
| `client.html` | **Страница для клиентов** — тарифы, устройства, рефералка |
| `landing.html` | Продажная страница (B2B) — продаём установку сервиса |
| `full-guide.html` | Полный гид: анализ рынка, финансы, инструкция установки |
| `client-delivery.html` | Документ сдачи клиенту (заполняется при установке) |
| `install-full.sh` | **Автоустановщик** — один скрипт ставит всё на сервер |
| `check.sh` | Проверка готовности системы перед сдачей |
| `bot/` | Python-бот (6 файлов, ~1100 строк) |
| `index.html` | Редирект на client.html для GitHub Pages |

---

## Стек

- **Marzban** — панель управления VPN-пользователями (Docker)
- **VLESS Reality** — протокол, маскируется под HTTPS, не блокируется в РФ
- **python-telegram-bot 20+** — async Telegram-бот
- **APScheduler** — фоновые задачи (проверка оплат, напоминания)
- **SQLite** — хранение пользователей, подписок, платежей
- **YooKassa** — карты и СБП
- **CryptoBot** — USDT/TON/BTC
- **Telegram Stars** — встроенная оплата без регистрации

## Быстрый старт (установка на VPS)

```bash
# 1. Скачать установщик
wget https://raw.githubusercontent.com/irekgai111-lang/vpn-sell/master/install-full.sh

# 2. Запустить (Ubuntu 22.04, root)
bash install-full.sh
# Скрипт интерактивно спросит: домен, BOT_TOKEN, ADMIN_ID, токены оплаты

# 3. Проверить готовность
bash check.sh
```

## Настройка client.html

Открой `client.html`, найди блок `CFG` в конце файла и замени:

```js
const CFG = {
  botName:  'YOUR_BOT',     // ← username твоего бота без @
  siteName: 'FastVPN',      // ← название сервиса
  siteUrl:  'https://...',  // ← URL этого сайта
};
```

## Команды администратора (в боте)

| Команда | Действие |
|---|---|
| `/stat` | Статистика: клиенты, выручка, активные подписки |
| `/adddays USER 30` | Добавить дни пользователю |
| `/broadcast TEXT` | Сообщение всем клиентам |
| `/promo CODE 20% 50` | Создать промокод на скидку 20%, 50 использований |
