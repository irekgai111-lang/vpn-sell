🛠️ Установка — как для пятиклассника
1
Деньги

Купи сервер (VPS)

Заходи на hetzner.com → Cloud → Add Server → Location: Ashburn USA → OS: Ubuntu 22.04 → Type: CAX11 (ARM, €3.29/мес) → SSH Key или пароль → Create. Запиши IP адрес сервера (например 5.42.123.45).

2
Подключение

Подключись к серверу

Windows: скачай MobaXterm → Session → SSH → вставь IP → Enter. Mac: открой Терминал и напиши: ssh root@ТУТ_IP → Enter → вводи пароль.

3
Автоустановка

Установи Marzban (одна команда)

Скопируй и вставь в терминал — нажми Enter и жди 5 минут:

sudo bash -c "$(curl -sL https://github.com/Gozargah/Marzban-scripts/raw/master/marzban.sh)" @ install

Скрипт сам установит всё. В конце спросит логин/пароль администратора — придумай и запиши!

4
Настройка

Открой панель управления

В браузере открой: http://ТУТ_IP:8000/dashboard

Войди с логином/паролем который задал при установке. Ты увидишь панель управления пользователями.

5
Протокол

Добавь VPN-протокол (VLESS Reality)

В панели: Hosts → Add Inbound → выбери VLESS → выбери Reality → укажи порт 443 → сохрани. Reality — самый надёжный протокол, не блокируется в России.

6
Telegram

Создай Telegram-бота для клиентов

В Telegram найди @BotFather → /newbot → придумай имя (например "Мой VPN") → придумай логин (например "myvpn2024_bot") → скопируй токен. Запиши токен — он выглядит как "7234567890:AAHxxxx..."

7
Связка

Подключи бота к Marzban

В терминале сервера: marzban cli set-setting --telegram-api-token "ТУТ_ТОКЕН"

Потом: marzban cli start-bot

Всё — бот запустился! Клиенты могут зарегистрироваться и получить ключ через бота.

8
Оплата

Подключи оплату через @CryptoBot

В Telegram найди @CryptoBot → Pay → Create App → получи API-токен. Поддерживает: USDT, TON, BTC, ETH. Деньги приходят тебе на крипто-кошелёк моментально. Комиссия 1%.

Для рублёвой оплаты — YooKassa (yookassa.ru) или Robokassa.

9
Бот продаж

Установи бота-продавца (скрипт готов)

Скопируй файл bot/bot.py из этого проекта на сервер. Настрой .env с токенами. Запусти. Бот будет сам принимать деньги → создавать ключи → отправлять клиентам.

pip install python-telegram-bot requests && python3 bot.py
10
Запуск

Запусти продажи

Поделись ссылкой на бота в соцсетях, ВКонтакте, Telegram-группах. Рекомендуй друзьям. Можно настроить рекламу в Telegram. Первые 10 клиентов — поделись бесплатно, получи отзывы.

❓ Часто задаваемые вопросы
QЭто законно — продавать VPN в России?
Да. Продажа VPN-сервисов частным лицам не запрещена. Запрещено только обходить блокировки самостоятельно на корпоративном уровне. Миллионы людей пользуются VPN — это нормально.
QСколько нужно вложить на старте?
Минимум — ~500₽ на первый месяц VPS. Больше ничего платить не нужно. Marzban бесплатный, Python бесплатный, Telegram-бот бесплатный.
QЧто если сервер упадёт?
Marzban включает встроенный watchdog. Плюс Hetzner даёт 99.9% uptime. При желании можно добавить второй сервер как резервный — клиенты автоматически переключатся.
QНужны технические знания?
Базовые — да. Скопировать команду в терминал, зайти на сайт, создать бота в @BotFather. Это уровень "пятиклассника". Если что-то пойдёт не так — ошибка видна в терминале, гугл + ChatGPT решают 90% проблем.
QМогу ли я принимать рубли?
Да. Через YooKassa, Robokassa, или просто по переводу на Сбербанк (но вручную). CryptoBot — самый удобный вариант (USDT стабильная монета, легко меняется на рубли через P2P).

🚀 Готов запустить?

Весь код для установки сервера и бота-продавца — в файлах install-marzban.sh и bot/bot.py этого проекта. Запусти скрипт на сервере — и через 10 минут у тебя работающий VPN-бизнес.
