Что у них есть, чего нет у нас:

1
Подготовка

Создай Telegram-бота

В Telegram найди @BotFather → напиши /newbot → придумай имя (например «Мой VPN 🔐») → придумай логин (например myvpn2026_bot) → скопируй токен (выглядит как 7234567890:AAH...). Сохрани в заметки.

Также узнай свой Telegram ID: найди @userinfobot → нажми Start → скопируй «Id:» число. Это твой ADMIN_ID.

2
Подготовка

Купи VPS-сервер

Заходи на hetzner.com → Cloud → Add Server:

• Location: Ashburn, VA (США, стабильный) или Falkenstein (Германия)
• Image: Ubuntu 22.04
• Type: CAX11 (ARM, 4 vCPU, 8 GB, 20 TB трафика — €3.79/мес ≈ 360₽)
• SSH Keys: если не знаешь что это — выбери «Password» и придумай пароль root
• Нажми «Create & Buy Now»

Через 30 секунд в письме придёт IP-адрес сервера. Запиши его (например 5.42.123.45).

3
Подготовка

Подключись к серверу

Windows: Скачай MobaXterm (mobaxterm.mobatek.net, бесплатная) → Session → SSH → Remote host = твой IP → OK → вводи пароль root (символы не видны — это нормально).

Mac/Linux: Открой Терминал → введи:

ssh root@5.42.123.45 # вместо 5.42.123.45 — твой IP

Напечатай пароль → Enter. Ты внутри сервера.

4
Установка

Скачай и запусти установщик

Скопируй этот блок целиком, вставь в терминал сервера, нажми Enter:

# Скачиваем проект
git clone https://github.com/irekgai111-lang/vpn-sell /opt/vpn-project 2>/dev/null || \
  wget -q -O /tmp/vpn.zip https://github.com/irekgai111-lang/vpn-sell/archive/main.zip && \
  unzip -q /tmp/vpn.zip -d /opt/ && mv /opt/vpn-sell-main /opt/vpn-project

# Запускаем установщик
cd /opt/vpn-project && bash install-full.sh

Скрипт задаст вопросы: токен бота, твой Telegram ID, домен (необязательно), ключи оплаты. Всё можно ввести сейчас или добавить позже. Установка занимает 5–7 минут.

5
Настройка

Настрой VLESS Reality в панели Marzban

Открой в браузере: http://ТВОй-IP:8000/dashboard
Войди с логином admin и паролем из терминала (написан в конце установки).

В панели: Хосты (Hosts) → Add Inbound → заполни:

• Протокол: VLESS
• Порт: 443
• Security: Reality
• Dest: www.microsoft.com:443
• SNI: www.microsoft.com
• Нажми «Generate» для Private Key и Short ID
• Tag/Name: vless-in (важно — это название используется в боте)

Сохрани. Теперь VPN-протокол активен.

6
Настройка

Подключи Telegram Stars (5 минут, бесплатно)

Это самый простой способ оплаты — клиенты платят звёздами Telegram.

В @BotFather → выбери своего бота → Bot Payments → Connect Payments Provider → выбери Telegram Stars → готово.

Всё — Stars работает без ИП и без банка. Добавь в конфиг бота:

nano /opt/vpn-bot/.env # добавь строку:
STARS_ENABLED=true

Перезапусти бота: systemctl restart vpn-bot

7
Настройка

Подключи карточную оплату — YooKassa

YooKassa принимает карты, СБП, ЮMoney. Нужно ИП или самозанятость.

1. Зарегистрируй самозанятость: npd.nalog.ru (мобильное приложение, 5 минут)
2. Регистрация в YooKassa: yookassa.ru → Подключиться → Физ. лицо/самозанятый
3. Загрузи паспорт (верификация 1–3 дня)
4. В личном кабинете: Интеграция → API → скопируй ShopID и Секретный ключ

nano /opt/vpn-bot/.env
YK_SHOP_ID=123456
YK_SECRET=live_xxxxxxxxxxxxxxxxxxxx

systemctl restart vpn-bot
8
Настройка

Подключи CryptoBot (криптоплатежи)

В Telegram найди @CryptoBot → Pay → Crypto Pay → Create App → дай название → скопируй API токен.

Принимает: USDT, TON, BTC, ETH. Комиссия 1%. Вывод моментально на кошелёк.

nano /opt/vpn-bot/.env
CRYPTO_TOKEN=токен_из_cryptopay

systemctl restart vpn-bot
9
Тест

Проверь что всё работает

1. Открой своего бота в Telegram → нажми /start
2. Нажми «Попробовать 3 дня бесплатно» → должен прийти VPN-ключ
3. Установи Hiddify на телефон → добавь ключ → включи → открой instagram.com
4. Попробуй купить платный план через Stars (если настроен)

Проверь логи бота если что-то не работает:

journalctl -u vpn-bot -f -n 50
