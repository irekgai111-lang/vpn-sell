Частые вопросы

Использование VPN для личных целей — закона, запрещающего это в России, нет. Десятки миллионов россиян пользуются VPN ежедневно. Мы не занимаемся нелегальной деятельностью и не помогаем ей.
Используем протокол VLESS Reality — он притворяется обычным HTTPS-сайтом (microsoft.com). Роскомнадзор не может отличить наш трафик от легитимного корпоративного. Именно поэтому его используют крупные VPN-сервисы в России.
Одна подписка — один VPN-ключ. Ключ можно добавить в приложение Hiddify сразу на телефоне, планшете, ноутбуке — до 5 устройств одновременно без доплат.
Бот напомнит за 3 дня и за 1 день до окончания. Откроешь — нажмёшь «Продлить» — оплатишь. Дни добавятся к текущей дате истечения, ключ останется тот же.
Напиши боту /help или кнопку «Поддержка» в меню. Чаще всего помогает: переустановить ключ, выбрать другой сервер в Hiddify, проверить что подписка не истекла (/mysub).

Инструкция по установке

📱 iPhone / iPad

1. Скачай Hiddify из App Store (бесплатно)

2. Открой приложение → «Добавить профиль»

3. Вставь ключ vless://... из бота

4. Нажми «Подключиться» — готово ✓

🤖 Android

1. Установи Hiddify из Google Play

2. «+» → «Вставить из буфера обмена»

3. Вставь ключ из бота → «Подключиться»

4. Разреши VPN-подключение при запросе

🍎 Mac

1. Скачай Hiddify с GitHub (файл .dmg)

2. Перетащи в «Программы», запусти

3. «Добавить профиль» → вставь ключ

4. Иконка в меню-баре → «Подключить»

🪟 Windows

1. Скачай Hiddify с GitHub (файл .exe)

2. Установи, запусти от имени администратора

3. «Добавить конфиг» → вставь ключ

4. Нажми «Старт» — иконка в трее станет зелёной ✓

🐧 Linux

1. Скачай .AppImage с GitHub

2. chmod +x Hiddify*.AppImage && ./Hiddify*.AppImage

3. Вставь ключ → подключайся

Политика конфиденциальности

Что мы собираем

При регистрации: Telegram ID и имя пользователя. Данные карт не хранятся — оплата через YooKassa, CryptoBot и Telegram Stars.

Как используем

Telegram ID нужен для: идентификации подписки, выдачи VPN-ключей, уведомлений о продлении. Мы не передаём данные третьим лицам и не продаём их.

Логи трафика

Мы не ведём логи посещённых сайтов, IP-адресов назначения или содержимого трафика. Хранится только объём потреблённого трафика (ГБ) для контроля лимита тарифа.

Хранение данных

Данные хранятся на серверах в Европе. Срок — до удаления аккаунта + 30 дней.

Удаление данных

Напиши боту /support с запросом на удаление — выполним за 3 рабочих дня.
